# Orbit Insights

**Space, Subcultures & The Future** — Exploring the intersection of space technology, emerging subcultures, and human expansion into the cosmos.

## About This Project

Orbit Insights is a newsletter and blog dedicated to deep analysis of space trends, new subcultures forming around space tech, and the societal impact of humanity becoming multi-planetary.

- **Website**: https://github.com/andrew1014/Orbit-Insights
- **Newsletter**: Delivered via MailerLite
- **Content Style**: Thoughtful, insightful, slightly optimistic with a touch of wonder

## Project Goals

- Publish high-quality weekly newsletter posts
- Build a growing archive of evergreen content
- Create an automated publishing pipeline
- Grow a dedicated subscriber list

## Tech Stack

- **Static Site**: Jekyll + GitHub Pages
- **Content**: Markdown files in `_posts/`
- **Automation**: GitHub + Custom webhook → MailerLite
- **AI Agent**: Hermes / APEX (local)

## Content Guidelines

- **Focus**: Space tech, subcultures, futurism, exploration
- **Tone**: Intelligent, accessible, optimistic but realistic
- **Length**: ~2500–3000 words per main newsletter
- **Include**: Strong hooks, clear sections, calls to action

## For Hermes / APEX Agent

- Always check `_posts/` folder for new content
- Follow daily session protocol
- Maintain consistent style across posts
- Use proper Jekyll frontmatter
- Prioritize automation and reliability

## Repository Structure

```yaml
/
├── _posts/                 # Markdown content files
├── assets/                # Static assets
├── _layouts/              # Jekyll layouts
├── _includes/             # Reusable HTML fragments
├── _data/                 # Configuration data files
├── index.md               # Homepage
└── (other Jekyll site files)
```
